# Portfolio-Web-React

Una aplicación web moderna y responsiva construida con React para mostrar proyectos de desarrollo y habilidades profesionales.

## 🚀 Características Clave
- **Galería de Proyectos**: Visualización dinámica de diversos proyectos de software y seguridad.
- **Diseño Responsivo**: Optimizado para visualización en móviles, tablets y escritorio.
- **UI/UX Moderna**: Estética limpia con transiciones e interacciones fluidas.

## 🛠️ Stack Tecnológico
- **Framework**: React.js.
- **Código Fuente**: Disponible en la carpeta `codigo_fuente/`.
- **Estilos**: CSS/SCSS estructurado en `recursos/`.
- **Gestión de Estado**: React Hooks (useState, useEffect).
- **Herramientas**: Vite/Webpack para builds optimizados.
